# c4_sem1_course_project

Чтобы запустить приложение, нужно в терминале прописать команду
```streamlit run app.py```   
В браузере откроется вкладка веб-приложения.      
![изображение](https://github.com/user-attachments/assets/c6a74c39-346c-4bd9-992a-f7ceecee200a)   
Для загрузки изображения нужно нажать кнопку "Browse files" и выбрать нужный файл. Загруженное изображение будет выведено на экран.   
![изображение](https://github.com/user-attachments/assets/2fc46df2-b7e2-452a-86d9-7061edd37026)    
Чтобы получить результат, нужно нажать кнопку "Классифицировать". Будет выведено исходное изображение и изображение с результатом работы модели.   
![изображение](https://github.com/user-attachments/assets/d4b04ebc-7480-4538-92ba-26ef967e872a)
